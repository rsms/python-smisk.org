debian

Smisk is available as Debian Linux packages from the Hunch repository, signed with the key . Replace stable|testing|unstable with the Debian dist of choice. 

deb http://hunch.se/debian/ stable|testing|unstable main
deb-src http://hunch.se/debian/ stable|testing|unstable main

The distributions "stable", "testing" and "unstable" refers to packages able to install on "stable" debian releases, "testing" debian releases and "unstable" debian releases, respectively. (i.e. not stable/testing/unstable versions of Smisk)
